# Agrotech_v1

Agrotech_v1 is a smart farming web application designed to help farmers monitor soil moisture, temperature, humidity, and environmental data in real time.  
The dashboard uses Firebase for authentication, real-time database operations, hosting, and analytics.  
It provides an easy-to-use and visually modern interface for smart agriculture monitoring.
